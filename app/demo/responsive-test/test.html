現在のモード: ウィンドウ幅を変えて確認

Responsive Logic

ご要望の「限界まで横並びを粘る」ロジックを実装しました。
ブラウザの幅をゆっくり縮めてみてください。

1. 通常モード (~1300px)

ナビ240px、画像500px。文章は最大720px。
ゆったりとしたレイアウトです。

2. レールナビモード (1080px ~ 1300px)

文章の幅を確保するため、まずナビが72px（アイコンのみ）に縮小します。
画像は500pxのままです。

3. 画像縮小モード (992px ~ 1080px)

ここが粘りポイント！
文章エリアが500pxを切りそうになると、画像が自動的に 420px に縮小します。
これにより、iPad ProやAirの横持ちなどでも、快適な文章幅を維持したまま横並び表示を続けられます。

4. 縦積みモード (992px以下)

これ以上は文章が読みづらくなるため、縦積みに切り替わります。
