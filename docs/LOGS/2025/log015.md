## SNSリンク設定仕様 (2025-06-22)

### 機能概要
- ユーザーが自分のSNSアカウントやWebサイトのリンクを設定・管理
- 管理者が用意した選択肢から選択 + カスタムリンクも対応
- アイコンは管理者用意またはユーザーアップロード可能
- ドラッグ&ドロップで並び替え可能

### データベース設計

#### LinkTypeテーブル (管理者管理)
```prisma
model LinkType {
  id           String @id @default(cuid())
  name         String @unique // "YouTube", "X", "Instagram", "Custom"
  displayName  String // 表示用名称
  defaultIcon  String? // デフォルトアイコンのパス
  urlPattern   String? // バリデーション用正規表現
  isCustom     Boolean @default(false) // カスタムリンクかどうか
  isActive     Boolean @default(true) // 利用可能かどうか
  sortOrder    Int @default(0) // 管理画面での表示順序
  createdAt    DateTime @default(now())
  updatedAt    DateTime @updatedAt
  
  userLinks    UserLink[]
}
```

#### UserLinkテーブル (ユーザーのリンク)
```prisma
model UserLink {
  id           String @id @default(cuid())
  userId       String
  linkTypeId   String
  url          String
  customLabel  String? // カスタムリンクのラベル
  customIconId String? // ユーザーアップロードアイコン
  sortOrder    Int @default(0) // ユーザー内での表示順序
  isVisible    Boolean @default(true) // 表示/非表示
  createdAt    DateTime @default(now())
  updatedAt    DateTime @updatedAt
  
  user         User @relation(fields: [userId], references: [id], onDelete: Cascade)
  linkType     LinkType @relation(fields: [linkTypeId], references: [id])
  customIcon   Media? @relation(fields: [customIconId], references: [id])
}
```

#### アップロードファイル検証
- **既存システムに準拠**: docs/GUIDES/IMAGE-UPLOADER-GUIDE.md に従う
- **対応形式**: gif, jpg, jpeg, png, webp, svg
- **SVGサニタイズ**: DOMPurify + JSDOM による自動サニタイズ
- **ファイルサイズ**: デフォルト制限に従う

- **表示場所**: ユーザーの個別ページ（未実装）

### セキュリティ（簡略版）
- **URL制限**: HTTPS必須のみ
- **有効性チェック**: 実装しない
- **カスタムリンク**: 制限なし（必要時に後から対策）
